# Oldal frissítése

Az oldal tartalomfrissítéséhez az alábbi GitHub platformot használjuk. Kövesd az alábbi lépéseket a tartalom szerkesztéséhez:

<Steps>

### Nyitás

Látogass el a MűhelyDokumentáció™ GitHub repository-jához: [megnyitás](https://github.com/barnagoz/eam)

Ha még nincs GitHub fiókod, hozz létre egyet a GitHub webhelyen.

### Forkolás

A repository oldalán kattints a "Fork" gombra a repository másolásához a saját fiókodba.

### Klónozás

Klónozd le a repository-t a saját gépedre, és módosítsd a tartalmat. Az oldal tartalmát a Markdown formátumban írt fájlok tartalmazzák.

### Pusholás

Amikor elkészültél a módosításokkal, push-olj (feltölts) minden változást a GitHub repository-ba.

### Pull request

Az utolsó lépés az úgynevezett pull request (PR) létrehozása. Ebben a lépésben jelezheted a végrehajtott módosításokat, és a repository tulajdonosai áttekinthetik azokat. Ezután eldönthetik, hogy beillesztik-e azokat az eredeti oldalba.

</Steps>

Reméljük, hogy könnyen megtalálod a szükséges információkat és aktívan részt veszel a MűhelyDokumentáció™ fejlesztésében!
